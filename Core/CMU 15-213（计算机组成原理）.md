### ✅ **Expanding（扩展，例如 short int → int）**

- **Unsigned（无符号）**：高位补零
- **Signed（有符号）**：进行符号扩展（符号位扩展到高位）
- ✅ **两者都能产生符合预期的结果**

📌 举例：

```c
unsigned short us = 0x00FF;
int i1 = us; // 结果是 0x000000FF

short s = -1;
int i2 = s;  // 结果是 0xFFFFFFFF（符号扩展）
```

------

### ✅ **Truncating（截断，例如 unsigned int → unsigned short）**

- **无符号/有符号**：只是简单截掉高位比特（bits are truncated）
- **结果被重新解释**：只保留目标类型能容纳的低位
- **Unsigned**：相当于执行模运算（mod operation），如 `value % 2^16`
- **Signed**：类似模运算，但解释方式不同（考虑符号位）
- ✅ **对于小数字，会表现为预期行为**

📌 举例：

```c
unsigned int x = 0x12345678;
unsigned short y = x; // y = 0x5678，相当于 x % 2^16

int z = -1;
short w = z; // w = -1（如果 short 是 16 位，结果是 0xFFFF）
```
